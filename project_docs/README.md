# 项目文档索引

本目录收纳本次任务相关的所有项目文档，作为训练侧工作的统一入口。

## 文档列表

- `原始研究动机.md`：保留最初的研究动机与问题分析原文。
- `研究摘要.md`：把研究方向压缩成更稳定的任务定义。
- `单人工作流.md`：约束 solo 模式下的推进方式。
- `任务计划.md`：维护当前阶段、约束、里程碑和下一步。
- `训练运行手册.md`：记录训练机上的实际执行方案。
- `实验进展.md`：记录每次重要进展、阻塞和后续动作。
- `论文细节参考.md`：摘录与当前实验最相关的论文训练细节。

## 使用约定

- 有新的关键决策时，优先更新 `任务计划.md`。
- 有新的实验执行、问题定位或结论时，优先更新 `实验进展.md`。
- 有研究方向变化时，同时更新 `原始研究动机.md` 或 `研究摘要.md`。
- 聊天记录不作为事实来源，仓库内文档才是事实来源。
